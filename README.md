# reactWeather
A simple weather app using React JS and Redux.Created following <a href="https://github.com/StephenGrider">StephenGrider</a> react with redux tut.
## Purpose
Display weekly average weather, pressure, and humidity.
## Use
```
	> cd reactWeather
	> npm install
	> npm start
```
